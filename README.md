
# toto-hint

"toto-hint"は,サッカーの勝敗予測可視化アプリケーションです.

<!-- ## DEMO -->

https://user-images.githubusercontent.com/65605945/147016596-1c93f42e-0087-4d46-937e-9c45e648ded0.mov


## Features

機械学習手法ランダムフォレストによる勝敗の予測確率,ポアソン分布による得点予測とそれに基づく勝敗の予測確率を可視化しています.
